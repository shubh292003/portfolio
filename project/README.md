# Portfolio Website

A premium portfolio website built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Modern Glassmorphism Design**: High-end visuals with backdrop blurs and subtle borders.
- **Premium Hero Section**: Showcasing a diverse tech stack (C++, Python, JS, etc.) with staggered animations.
- **Interactive Navbar**: Sticky floating island navigation with custom-styled tooltips.
- **Responsive Layout**: Designed to look great on all devices.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## How to Run
Simply open `portfolio.html` in any modern web browser.
